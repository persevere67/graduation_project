# 基于联邦学习的隐私保护新闻推荐算法研究

本项目为毕业设计实验代码仓库。

## 1. 项目简介
使用 **MIND** 数据集，基于 **Flower** 联邦学习框架，实现了结合 **SBERT** 新闻编码与 **Multi-Head Attention** 用户编码的双塔推荐模型。

## 2. 实验进展
- [x] 数据预处理与特征提取 (SBERT)
- [x] 中心化 Baseline 训练 (Loss: 1.31)
- [x] 联邦学习环境搭建 (Flower)
- [x] 联邦学习模型收敛实验 (50 Clients, 10 Rounds, Loss: 1.515)

## 3. 运行环境
- Python 3.11+
- PyTorch
- flwr (Flower)
- sentence-transformers

## 4.初步思路
毕业论文大纲（CQU 标准版）
第 1 章 绪论
1.1 研究背景与意义：信息爆炸时代对个性化新闻推荐的需求；传统中心化推荐面临的隐私合规风险（如数据隐私法案）。

1.2 国内外研究现状：

1.2.1 新闻推荐算法现状（从协同过滤到深度学习）。

1.2.2 联邦学习（Federated Learning）在工业界的应用现状。

1.3 本文主要研究内容：

结合 SBERT 与多头自注意力机制的双塔推荐模型设计 。

基于 Flower 框架的隐私保护分布式训练方案 。

1.4 论文组织结构。

第 2 章 相关技术综述
2.1 自然语言处理技术：SBERT (Sentence-BERT) 的特征提取原理。


2.2 推荐模型核心机制：多头自注意力机制 (Multi-Head Attention) ；点积相似度计算。

2.3 联邦学习理论：FedAvg 聚合算法；“数据不动模型动”的隐私保护逻辑。


2.4 开发环境与工具：Python 3.11, PyTorch, Flower, RTX 3060 硬件环境 。

第 3 章 基于注意力机制的新闻推荐模型设计（核心）

3.1 总体架构设计：双塔结构模型原理（User Tower & News Tower） 。

3.2 新闻编码器设计：SBERT 预训练向量的特征投影与微调。

3.3 用户编码器设计：

3.3.1 历史点击序列处理 。

3.3.2 Multi-Head Self-Attention 捕捉用户动态兴趣关联 。

3.3.3 残差连接与归一化（Add & Norm） 。


3.4 匹配与排序模块：基于余弦相似度/点积的点击概率预测 。

第 4 章 隐私保护下的联邦学习训练方案（核心）

4.1 联邦学习系统建模：服务器（Server）与 50 个模拟客户端（Clients）的拓扑结构 。

4.2 本地训练过程：

4.2.1 本地负采样策略 。

4.3 全局参数聚合：FedAvg 策略的具体实现与参数交换流程 。


4.4 隐私安全性分析：分析为何不直接传输用户历史点击记录（History）能实现隐私保护 。
